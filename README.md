Este projeto consiste em uma API REST para cadastro e gerenciamento de usuários, desenvolvida com foco em boas práticas de backend e organização de código. O objetivo é simular um sistema real utilizado em aplicações corporativas.

## Funcionalidades
- Cadastro, consulta, atualização e remoção de usuários (CRUD)
- Validações básicas dos dados de entrada
- Uso de DTOs para entrada e saída de dados
- Tratamento global de exceções com `@RestControllerAdvice`

## Tecnologias utilizadas
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven ou Gradle
- Git e GitHub
